# Features from Minecraft 1.21

Java **1.21** shipped the **Crafter**, many **copper** blocks, **wind charges**, **buffed witch
redstone drops**, and more. **Redstone Backport** currently ships:

- **Crafter** (autocrafting block; redstone pulse; comparator; hopper/dropper interaction; disabled
  slots).
- **Witch loot:** **4–8 redstone dust** (with looting), aligned with the 1.21 farm buff.

:::{tip}
Comparator output, pulse crafting, and hopper automation follow **1.21** design goals; for precise
numbers, test in-game on your loader.
:::

## Crafter

The Crafter crafts when given a **redstone pulse**, can **eject** results directionally, supports
**disabled slots**, works with **hoppers** and **droppers**, and exposes **comparator** strength
based on configured / disabled slots (vanilla uses 0–9 style signaling).

Implementation lives in the `common` module with **Forge** / **Fabric** / **Quilt** registration for
block, block entity, menu, and screen.

## Witch redstone drops

Witches drop **4–8 redstone dust** (subject to looting), consistent with **1.21** release notes on
improving witch-farm renewability.

:::{note}
**Loader wiring**

- **Fabric / Quilt:** loot table event modifies `minecraft:entities/witch`.
- **Forge:** global loot modifier registered in mod metadata.
:::

## Not backported from 1.21 (today)

The following appear in the redstone changelog for **1.21** but are **not** in this mod yet. See the
{ref}`version matrix <version-matrix>`.

:::{note}

- Copper **Bulb** (pulse memory, comparator 15 when lit)
- Copper **doors** / **trapdoors**
- **Wind charges** from dispensers; wind burst interactions
- Reduced **piston** sound volume (QoL only)
:::

Future releases may expand this list; the matrix on the home page summarizes what is in versus out.
