# Feature reference

Deep dives for **Minecraft Java** releases this mod draws from. Each page summarizes **vanilla**
behavior, what **Redstone Backport** ships on **1.20.1**, and implementation or **protocol** notes
where it matters.

The **version matrix** on the {ref}`home page <version-matrix>` is the high-level checklist; these
pages are the detail.

## Features from Minecraft 1.20.3

Vanilla **1.20.3** added **decorated pots**, the **`/tick`** command (freeze, step, rates, sprint),
and an **experimental Crafter** preview. This mod focuses on **`/tick`** and tick managers on
**1.20.1**; pots and the **1.20.3** crafter preview are **not** ported here (see the **1.21** Crafter
page instead).

→ {doc}`Minecraft 1.20.3 reference <v1-20-3>`

## Features from Minecraft 1.21

**Crafter** automation, **copper** gear, **wind charges**, **witch** redstone drops, and more
landed in **1.21**. The mod currently documents and ships **Crafter** + **witch loot** parity work;
other **1.21** items remain on the matrix as **Planned** or **Partial**.

→ {doc}`Minecraft 1.21 reference <v1-21>`
